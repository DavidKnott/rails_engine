## Setup instructions
Run these commands in this order:
```
git clone XXX
cd rails_engine
rails db:create
rails db:migrate
rails db:seed
rails db:test:prepare
```

## Deployment instructions
To launch the project on local host, start up the rails server and visit any of the routes listed below.
